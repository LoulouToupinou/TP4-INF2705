Animations:
    - On donne l'option à l'utilisateur de faire tourner l'épée sur elle même ou non
    - L'orbe est animée pour donner l'impression qu'elle flotte
    - On peut animer l'épée comme si on donnait un coup d'épée

E7:
    - Bézier: l'épée suit une trajectoire qui est une courbe de Bézier lorsqu'elle est animée
    - Phong: On utilise l'illumination de phong pour illuminer la scène. L'orbe emet de la lumière. On peut contrôler la lumière ambiante avec un slider ImGui

E8:
    - Bloom: On applique un effet de bloom à l'orbe pour qu'elle sois plus brillante
    - Correction gamma: On peut appliquer une correction gamma avec un slider ImGui
